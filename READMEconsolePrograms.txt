Examples of console IO programs can be found in 
   examples/consoleExamples/

The main library needed is in 
    src/ConsoleLib.agda

Note that there is no need to install wxWidgets and wxHaskell if you are 
only interested in console programs.

For compiling program use Agda-> Compile and type in as backend GHC.
If compiling the examples in examples/consoleExamples/ the executable programs
will be in the directory examples
